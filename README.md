# 📒 Recordly

## 🚀 About the Project
Recordly is a simple and efficient web-based record management system built using HTML, CSS, and JavaScript. It helps users manage and organize records in a clean and user-friendly interface.

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
