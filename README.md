# 📒 Recordly

## 🚀 About the Project
Recordly is a simple and efficient web-based record management system built using HTML, CSS, and JavaScript. It allows users to manage and organize records in a clean and easy-to-use interface.

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
